衛星影像索引系統


由於現今極端氣候及氣候變遷時常發生，從廣為人知的海平面上升、融冰現象、森林野火等等，一路到我們身邊天氣的炎熱甚至是前陣子的缺水事件，可以深刻的體會到氣候變遷對於我們的生活影響非常的大，同時地球環境受到氣候變遷的影響就彷彿癌症對於人體的影響，是相當緩慢的，常常都是在已經到了最後無法拯救挽回的地步才會被人發覺，這樣無論是企業還是政府都難以去做到相對應的永續發展之決策，因此若是有一個軟體可以迅速瞭解目標地區的環境變化甚至是氣候變遷的影響程度，即可讓企業或政府在制定永續發展或是環境保護的決策時，能夠更加快速及準確地做出正確的判斷。 

第一步:
透過使用者輸入的經緯度座標可以更加準確的找到想確認或是搜尋的地方影像。

第二步:
以搜尋到的該地區影像進入影像資料庫進行比對以及分析，這除了可以幫助使用者了解該區域的變化外，若同時有需要了解其他哪些地區有同樣情況發生的話，這步驟也可以提供其相對應的幫助。
